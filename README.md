# Noti2me - 時間割
スケジュール管理アプリ風のタイムラインを表示できる時間割アプリです。

## 機能
・今日これからある講義情報のタイムライン表示  
・講義開始前の講義情報の通知  
・出席状況の記録  
・講義ごとのTodoリスト  
・講義のメモ (画像添付可)
